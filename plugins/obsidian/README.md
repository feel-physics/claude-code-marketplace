# obsidian

Obsidian関連のClaude Codeスキル集。

## Skills

| Skill | Description |
|-------|-------------|
| show-dotfiles | Obsidianでドットフォルダ（`.claude` 等）を表示する |

## Install

```
/plugin marketplace add https://github.com/feel-physics/claude-code-marketplace.git
/plugin install obsidian@feel-physics
```
